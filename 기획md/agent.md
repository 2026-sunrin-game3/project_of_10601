# 보스 개요

이 게임에는 단일 보스가 등장한다.

보스전은 총 2개의 실제 전투 페이즈로 구성된다.

## Phase 1

보스가 네 가지 형태를 번갈아 사용한다.

- 기본 형태
- 소형 형태
- 대형 형태
- 특수 형태

## Phase 3

보스가 두 가지 형태를 번갈아 사용한다.

- 공격 형태
- 수비 형태

## 제외된 내용

기존 Phase 2 비행 형태는 현재 개발 범위에서 제외한다.

문서와 코드에서는 기존 명칭을 유지하기 위해 마지막 페이즈를 Phase 3으로 부를 수 있다.
다만 실제 게임 진행상으로는 두 번째 전투 단계이다.

## 기존 코드

이 프로젝트는 이미 일부 기능이 구현되어 있다.

새로운 기능을 만들기 전에 반드시 기존 코드를 분석한다.

이미 존재하는 시스템을 새로 구현하지 않는다.

가능한 한 기존 구조를 유지하며 필요한 부분만 수정한다.
